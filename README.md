# SpendSmart

## Overview
SpendSmart is a personal expense tracking application built with ASP.NET Core and Entity Framework Core.

## Features

✔ Add expenses
✔ Edit expenses
✔ Delete expenses
✔ View all expenses

## Technologies

- C#
- ASP.NET Core
- Entity Framework Core
- In-Memory Database

## Architecture

- Controllers
- Models
- Services
- Repository

## Future Improvements

- Authentication
- SQL Server
- Categories
- Monthly reports
- Search expense
